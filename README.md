## API ViaCEP - Laravel 10

Aplicação para cadastro de endereços utilizando a API ViaCEP.


## Instalação

Para instalar a aplicação, siga os passos abaixo:

```bash
git clone
```

```bash
composer install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```
```bash
php artisan migrate
```

```bash
php artisan serve
```
